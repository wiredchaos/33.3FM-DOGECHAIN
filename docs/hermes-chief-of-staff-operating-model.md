# Hermes Chief of Staff Operating Model

Status: architecture note
Scope: 33.3FM DOGECHAIN

## Core lock

33.3FM stays an independent radio studio lane.

Hermes can coordinate production work, but Hermes does not merge this lane into FEN, VAULT33, VRG33589, or 589 Magazine.

## Hermes role for 33.3FM

Hermes may help with:

- show planning
- episode briefs
- script drafts
- content queues
- asset requests
- publishing checklists
- repo documentation
- status reports

## Operating doctrine

1. Keep 33.3FM DOGECHAIN independent.
2. Use Hermes for coordination and repeatable workflows.
3. Keep approval gates on publishing.
4. Save stable rules into docs.
5. Keep canon boundaries clean.

## Production loop

1. Brief received.
2. Hermes checks brand and canon notes.
3. Draft is prepared.
4. Assets are queued.
5. NEURO approves.
6. Output is published or saved.
7. Decision is logged.

## Hard boundaries

- No FEN portal linking.
- No VAULT33 merge.
- No VRG33589 merge.
- No 589 Magazine merge.
- No Akashic bleed unless explicitly approved as a separate mirror.

## Anti drift note
